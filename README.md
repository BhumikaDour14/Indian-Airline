# ✈️ India Airline Webpage

A simple and responsive airline booking website built using HTML, CSS, and JavaScript.
Users can view available flights, book tickets, and see seat availability update in real-time after booking.

# 🚀 Features

- View available flights
- Book airline tickets
- Real-time seat reduction after booking
- Booking confirmation popup message
- Responsive and user-friendly design
- Simple frontend project using pure HTML, CSS & JavaScript
  
# 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
